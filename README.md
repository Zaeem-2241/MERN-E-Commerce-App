# 🛒 MERN E-Commerce App

A full-stack eCommerce web application built using the MERN stack with advanced features like authentication, admin dashboard, cart system, order management, and Stripe payment integration.

---

## 🚀 Features

### 👤 User Features
- User Registration & Login (JWT Authentication)
- Browse Products
- Search & Filter Products
- Add to Cart / Remove from Cart
- Place Orders
- View Order History

### 🔐 Admin Features (RBAC)
- Admin Dashboard
- Manage Products (Create, Update, Delete)
- Manage Users
- Manage Orders
- Update Order Status (Pending → Delivered)

### 🛍️ E-Commerce Features
- Product Pagination
- Infinite Scroll / Lazy Loading
- Product Details Page
- Stock Management
- Responsive UI (Mobile Friendly)

### 💳 Payment Integration
- Stripe Payment Gateway Integration
- Secure checkout process

---

### Frontend
- React.js

- Node.js

- MongoDB (Mongoose)
